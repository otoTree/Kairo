 - 功能闭环与发布质量达标
 - 文档、示例与验证体系完整

 - 全链路 E2E 验收场景

 - 版本升级与迁移流程可用

 - Agent 可稳定完成跨会话长任务
 - 记忆、权限、设备、技能链路全可用
 - 发布流程可复现并可回滚

- 跨会话长任务从回忆到交付完整闭环

- 单机多技能并发任务稳定运行
- 权限、审计与可观测性链路闭环
- 版本升级可回滚且不丢数据

## 非目标
- 生态规模化与商业化分发
