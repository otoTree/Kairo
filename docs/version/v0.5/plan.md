 # v0.5 版本规划（Skills 分发与运行形态）
 
 ## 目标
 - 技能分发与版本管理可用
 - 多形态技能运行路径一致
 
 ## 交付范围
 - artifacts 下载、校验、升级与回滚
 - binary/wasm/container 选择与隔离策略
 - 技能运行依赖的统一治理
 
 ## 验收标准
 - 同一技能可在不同平台选择正确制品
 - 技能升级失败可一键回滚
 - 运行时隔离策略稳定可观测

## 纵向切片（E2E）
- 完成技能安装、升级失败回滚与继续运行
- 运行形态差异不影响权限与审计链路

## 关键场景
- 技能升级失败时自动回滚到上一个版本
- 同一技能在 macOS/Linux 自动匹配制品
- 运行形态不同但权限与审计一致

## 非目标
- 完整市场与分发平台
- 跨节点分布式调度
- 复杂依赖图解析与冲突解决
