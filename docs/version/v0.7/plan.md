 # v0.7 版本规划（状态与恢复）
 
 ## 目标
 - 关键实体持久化与重启恢复
 - 事件投影与摘要可用
 
 ## 交付范围
 - ProcessSession/DeviceClaim 等核心实体持久化
 - 事件流投影与压缩策略
 - 重启恢复与补偿路径
 
 ## 验收标准
 - 重启后可恢复进行中的会话状态
 - 事件流可生成一致的视图投影
 - 中断任务具备明确的失败语义

## 纵向切片（E2E）
- 任务执行中重启并恢复到可继续状态
- 投影视图与事件回放一致

## 关键场景
- 重启后恢复进行中的长任务状态
- 通过投影快速回答任务当前进度
- 中断与取消具备可回放证据链

## 非目标
- 全量历史事件长期保留
- 复杂事务与强一致分布式存储
- 跨版本自动迁移所有旧数据
