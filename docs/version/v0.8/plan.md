 # v0.8 版本规划（交互层与协作）
 
 ## 目标
 - UI/CLI 与事件流稳定对接
 - 协作原语可用并具备权限边界
 
 ## 交付范围
 - Server Bridge 与前端事件订阅
 - 协作窗口与共享空间原语
 - 交互层的权限与审计对齐
 
 ## 验收标准
 - UI 仅依赖事件与流即可还原状态
 - 多窗口协作操作可回放
 - 协作能力具备最小权限控制

## 纵向切片（E2E）
- UI 侧仅订阅事件即可重建协作状态
- 协作操作链路可关联到意图与主体

## 关键场景
- 多人协作中 UI 可基于事件重建视图
- 协作操作可追溯到意图与权限主体
- UI 端无直接访问核心状态的后门

## 非目标
- 完整桌面系统与窗口管理器
- 实时多端同步的复杂冲突解决
- 自定义 UI 生态与插件市场
